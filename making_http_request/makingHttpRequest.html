JS request making

XMLHttpRequest

The Original way of sending requests via JS.

  • Least Useful Way out of all!
  • Does not support promises, so lots of callbacks!
  • Clunky Syntax (difficult to remember)

Fetch API

This allows us to make a request using a function called Fetch

  • The newer better way of making requests via JS. (Supports Promises)
  • Not supported in Internet Explorer.
